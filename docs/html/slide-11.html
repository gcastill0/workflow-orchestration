Additional definitions
  • Service orchestration is broad approach and seeks to provide a complete end-to-end solution for delivering a “service”
  • Cloud orchestration reflects the modern equivalent of infrastructure setup performed by system administrators with provisioning tools like Chef, Puppet, and others
  • Release orchestration is a combination of deployment automation, pipeline and environment management
  • Workflow orchestration defines a logical sequence of discrete activities from a Start event to an End event to accomplish a specific service
